## GoWftp是什么?
一个用Golang编写的支持多用户的FTP服务端，并支持通过WEB界面进行管理服务器。

## GoWftp有哪些功能？

*  多用户，每个用户可以有不同权限，不同用户可以公用同一目录且不同权限
*  WEB管理服务器
*  用户通过WEB管理FTP空间（未完成，请用FTP登录）
